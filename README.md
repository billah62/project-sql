# Training Center Management System (SQL)

This project is a SQL-based Training Center Management System designed to manage
students, courses, trainers, batches, enrollments, and payment records.

## Features
- Student registration and management
- Course and batch management
- Trainer assignment
- Student enrollment tracking
- Fee and payment management
- SQL queries for reporting and analysis

## Database Tables
- students
- courses
- trainers
- batches
- enrollments
- payments

## Technologies Used
- MySQL

## How to Use
1. Create a database in MySQL
2. Import the provided `.sql` file
3. Run queries to manage and analyze training center data
